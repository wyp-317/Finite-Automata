# FloatDFA: 浮点数有限状态自动机 (DFA) 校验器

## 📝 简介

本项目提供了一个基于 **确定有穷自动机 (Deterministic Finite Automaton, DFA)** 的 Python 实现，用于精确校验字符串是否符合合法的 C 语言风格浮点数模式。它可以识别包括正负号、整数部分、小数部分以及科学计数法（`e/E`）在内的复杂数值结构。

本项目是展示计算理论中**词法分析**及自动机模型在实际工程中应用的典型案例。

---

## 🧠 核心原理

本项目的理论基础是确定有穷自动机。在形式化定义上，该自动机由一个五元组 $M = (Q, \Sigma, \delta, q_0, F)$ 组成：

- **$Q$ (状态集)**：包含 8 个离散状态，代表数值解析的不同阶段。
- **$\Sigma$ (字母表)**：系统将输入字符映射为 4 种基础类型：数字 `[0-9]`、符号 `[+/-]`、小数点 `.` 及指数符号 `[e/E]`。
- **$\delta$ (转移函数)**：定义了在特定状态下接收特定字符后跳转的目标状态。
- **$q_0$ (初态)**：状态 0，即运算的起点。
- **$F$ (终态集)**：满足合法的数值结尾（状态 2, 4, 7）。

---

## ⚙️ 状态设计与逻辑控制

自动机通过维护一个“当前状态”变量来处理输入流，避免了复杂的正则表达式回溯，实现了 $O(n)$ 的线性时间复杂度。其内部状态转移逻辑如下表所示：

| 状态编号 | 含义描述 | 下一步可能的合法输入 (触发跳转) |
| :--- | :--- | :--- |
| **0** | **起始状态** | 数字 (->2), 符号 (->1), 点 (->3) |
| **1** | **符号位已读** | 数字 (->2), 点 (->3) |
| **2** | **整数部分** (⭐ **接受状态**) | 数字 (->2), 点 (->4), 指数 (->5) |
| **3** | **孤立小数点** (如 `.5` 前的点) | 数字 (->4) |
| **4** | **小数部分** (⭐ **接受状态**) | 数字 (->4), 指数 (->5) |
| **5** | **指数符号 (e/E)** | 数字 (->7), 符号 (->6) |
| **6** | **指数符号位 (+/-)** | 数字 (->7) |
| **7** | **指数数字部分** (⭐ **接受状态**) | 数字 (->7) |

*(注：只有当字符串遍历结束，且当前状态停留在 **2、4 或 7** 时，该字符串才被判定为合法浮点数。若遇到未定义的输入或不允许的转移，状态机将立即拒绝该输入。)*

## 🚀 运行与使用

### 1. 环境要求
- Python 3.x (仅依赖 Python 标准库，无需额外安装第三方包)

### 2. 快速启动
引入 `FloatDFA` 类，并调用 `validate(text)` 方法传入需要校验的字符串即可：

```python
# 引入 DFA 校验器
from your_file_name import FloatDFA

# 初始化自动机
dfa = FloatDFA()

# 校验合法输入
dfa.validate("+3.14")   # 输出: 接受
dfa.validate("1e-10")   # 输出: 接受
dfa.validate(".5")      # 输出: 接受

# 校验非法输入
dfa.validate("abc")     # 输出: 拒绝
dfa.validate("1.2.3")   # 输出: 拒绝
