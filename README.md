# 🤖 浮点数识别器 - 有穷自动机 (DFA)

本项目为编译原理课程的词法分析实践部分。我们通过手工构建**确定性有穷自动机 (DFA)**，实现对浮点数（包含整数、小数、科学计数法）字面量的精准识别。

## 🎯 设计原理与 DFA 结构

浮点数识别器的核心等价于正则表达式：`[+-]?(\d+(\.\d*)?|\.\d+)([eE][+-]?\d+)?`

### 字母表 (Alphabet, Σ)
我们将输入字符简化映射为 4 种类型：
- **`0`**: 数字 (`0-9`)
- **`1`**: 正负号 (`+`, `-`)
- **`2`**: 小数点 (`.`)
- **`3`**: 指数符号 (`e`, `E`)

### 状态集合 (States, Q)
- `q0` (0): 初始状态
- `q1` (1): 符号位（接收到正负号后）
- **`q2` (2): 整数部分** —— 【接受状态】
- `q3` (3): 起始小数点（如输入 `.5` 时，处理点号后的状态）
- **`q4` (4): 小数部分** —— 【接受状态】
- `q5` (5): 指数符号 (E/e)
- `q6` (6): 指数符号位 (E后面的正负号)
- **`q7` (7): 指数部分数字** —— 【接受状态】

### 状态转移表 (Transition Table, δ)
| 状态 \ 输入 | 0 (数字) | 1 (符号) | 2 (点) | 3 (E/e) |
| :--- | :---: | :---: | :---: | :---: |
| **q0 (初始)** | q2 | q1 | q3 | / |
| **q1 (符号)** | q2 | / | q3 | / |
| **q2 (整数)* **| q2 | / | q4 | q5 |
| **q3 (始点)** | q4 | / | / | / |
| **q4 (小数)* **| q4 | / | / | q5 |
| **q5 (E)** | q7 | q6 | / | / |
| **q6 (E符号)**| q7 | / | / | / |
| **q7 (E数字)***| q7 | / | / | / |

*(注：带*号的为接受状态 Accept States，`/` 表示非法转移，进入死状态 Reject)*
